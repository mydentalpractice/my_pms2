My Dental Plan Healthcare Pvt. Ltd. - New Features

When an appointment is created/rescheduled, MyDental Plan sends SMS/Email notification to the Provider and the attending Doctor. To improve the Provider's/Doctor's user experience, MyDental Plan will group these notifications into a single SMS/Email notification and send it only once. However, the Provider has the option to override grouping and set to receive individual notification when the appointment is created/rescheduled/cancelled. This document explains these new Features and how to configure them for individual Provider/Doctor.


NEW FEATURES:

  1. Group SMS : This feature groups all New/Rescheduled Appointment Notifications in a single SMS Notification to be sent to the Provider/Doctor
  2. Group Email : This feature groups all New/Rescheduled Appointment Notifications in a single Email Notification to be sent to the Provider/Doctor
  3. Individual SMS : This feature sends a single SMS Notification to the Provider/Doctor for each appointment that is created/rescheduled
  4. Individual Email: This feature sends a single Email Notification to the Provider/Doctor for each appointment that is created/rescheduled

The Provider has total flexiibily to turn on/off these features for each Doctor in the clinic. The following section explains how to configure these settings.

HOW TO?

  1. From the Dashboar/Home screen, select 'Settings' menu option
  2. From 'Settings' pop-up menu, select 'Clinic Staff Details' menu option
  3. When creating or editing a new Doctor, there are four check-boxes - Individual SMS, Individual Email, Group SMS, Group Email. You can check/uncheck individual check-boxes to turn on/off each feature.
